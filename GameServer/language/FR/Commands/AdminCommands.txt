
# /serverproperties Command

# /plvl Command
AdminCommands.plvl.Description:						Change le niveau de privilèges ou l'attribution de commande d'un joueur ou d'un compte de jeu.
AdminCommands.plvl.Usage:							/plvl <NouveauRang> [NomDuJoueur]
AdminCommands.plvl.Usage.Single:					/plvl single <commande> [NomDuJoueur]
AdminCommands.plvl.Usage.SingleAccount:				/plvl singleaccount <commande> [NomDuCompte]
AdminCommands.plvl.Usage.Remove:					/plvl remove <commande> [NomDuJoueur]
AdminCommands.plvl.Usage.RemoveAccount:				/plvl removeaccount <commande> [NomDuCompte]
AdminCommands.plvl.NoSelectPlayer:					Vous devez sélectionner un joueur.
AdminCommands.plvl.AddedSinglePermission:			Vous autorisez le joueur {0} à utiliser la commande: {1}.
AdminCommands.plvl.AddedSingleAccountPermission:	Vous autorisez le compte de jeu {0} à utiliser la commande: {1}.
AdminCommands.plvl.RemoveSinglePermission:			Vous retirez au joueur {0} le droit d'utiliser la commande: {1}.
AdminCommands.plvl.RemoveSingleAccountPermission:	Vous retirez au compte de jeu {0} le droit d'utiliser la commande: {1}.
AdminCommands.plvl.NoPermissionForCommand:			Il n'existe aucune autorisation pour {0} d'utiliser la commande {1}.
AdminCommands.plvl.YourPlvlHasBeenSetted:			Vos privilèges sont maintenant de niveau {0}
AdminCommands.plvl.PlayerPlvlHasBeenSetted:			Le niveau de privilèges du joueur {0} est maintenant: {1}
